# go-hdwallet

> HD Wallet derivations from [mnemonic] seed in Go (golang). Implements the [go-ethereum](https://github.com/ethereum/go-ethereum) [`accounts.Wallet`](https://github.com/ethereum/go-ethereum/blob/master/accounts/accounts.go) [`address.Address`](github.com/fbsobreira/gotron-sdk/pkg/address) interface.

[![License](http://img.shields.io/badge/license-MIT-blue.svg)](https://raw.githubusercontent.com/miguelmota/go-ethereum-hdwallet/master/LICENSE)

## Install

```bash
go get -u github.com/hinccvi/go-hdwallet
```
